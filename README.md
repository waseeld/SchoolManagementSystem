## ğŸ“˜ Project Overview
The **School Management System (SMS)** is a simple console-based Java application designed to manage a school's daily activities.  
It can store and display information about students, teachers, departments, classrooms, labs, buses, and more.

The goal of this project is to demonstrate **Object-Oriented Programming (OOP)** concepts such as:
- Classes and Objects  
- Inheritance  
- Abstraction  
- Composition and Aggregation  
- Association
